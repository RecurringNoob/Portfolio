# Netflix-Style Portfolio (Frontend Clone)

A Netflix-inspired developer portfolio showcasing projects in an interactive,
profile-based streaming UI.

## Features
- Profile selection flow
- Billboard hero section
- Horizontal scrolling rows
- Animated project modal
- Framer Motion transitions
- Responsive design
- Local state management

## Tech Stack
- React
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Architecture
- Component-driven design
- Custom hooks for scroll & UI state
- Data-driven rendering

## What I Learned
- Complex UI state management
- Animation orchestration
- Scalable component architecture
